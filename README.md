# 리듬PAD로 배우는 BEAT 프로그램 웹페이지

GitHub Pages에 바로 올릴 수 있는 정적 웹사이트입니다.

## 파일 구성

- `index.html`: 메인 웹페이지
- `style.css`: 디자인 스타일
- `script.js`: 설명서 미리보기 탭 기능
- `assets/`: 설명서 이미지와 PDF 파일

## GitHub Pages 배포 순서

1. GitHub에서 새 repository를 만듭니다.
2. 이 폴더 안의 모든 파일을 repository에 업로드합니다.
3. repository의 `Settings` 메뉴로 이동합니다.
4. `Pages` 메뉴에서 `Deploy from a branch`를 선택합니다.
5. Branch는 `main`, Folder는 `/root`를 선택하고 저장합니다.
6. 잠시 후 `https://GitHub아이디.github.io/repository이름/` 주소로 접속합니다.
