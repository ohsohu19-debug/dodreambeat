리듬 PAD로 배우는 BEAT프로그램

리듬PAD로 배우는 BEAT 프로그램은 장구와 전통 장단을 앱으로 탐색하고, 연습과 실전 활동, 합주 모드까지 연결해 수업에서 바로 활용할 수 있도록 구성한 학습 프로그램입니다.

교사용 설명서 학생용 설명서·학습지

Program Overview

무엇을 배울 수 있나요?

01

장구 알아보기

장구의 구조와 명칭, 부편과 채편의 소리 차이를 시각 자료와 함께 익힙니다.

02

장단 알아보기

휘모리, 굿거리, 자진모리, 중중모리, 세마치 등 주요 장단을 듣고 따라 합니다.

03

장단 연습하기

난이도와 속도를 조절하며 리듬PAD로 정확한 박과 순서를 연습합니다.

04

함께 합주하기

같은 네트워크 안에서 방을 만들고 친구들과 같은 장단을 함께 연주합니다.

Class Flow

수업에서 이렇게 활용하세요

  1. 도입 리듬PAD 연결 상태를 확인하고 장구의 구조와 소리 원리를 살펴봅니다.
  2. 전개 1 장단의 입보, 구음, 동작 영상을 보며 기본 흐름을 익힙니다.
  3. 전개 2 연습 모드와 실전 장단치기로 개인별 정확도와 박자감을 높입니다.
  4. 정리·확장 합주 모드와 리듬 메이커로 협력 연주와 창작 리듬 활동을 진행합니다.

GitHub Pages

무료 공개 방법

1. 새 저장소 만들기

GitHub에서 새 repository를 만들고 이름을 예: dodreambeat로 지정합니다.

2. 파일 업로드

이 폴더의 index.html, style.css, script.js, assets 폴더를 업로드합니다.

3. Pages 켜기

Settings → Pages → Deploy from a branch → main / root를 선택합니다.

4. 주소 공유

잠시 후 https://아이디.github.io/dodreambeat/ 형식의 주소가 생깁니다.
